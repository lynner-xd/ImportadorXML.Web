API Pública de Integração

A API de integração do Select Contábil permite que o sistema da sua empresa importe XMLs fiscais, crie e consulte lançamentos contábeis e obtenha os relatórios contábeis em JSON ou PDF.

URL base:

{{ baseUrl }}

Autenticação

Todas as chamadas exigem o header X-Api-Key com o token fornecido pelo seu contador. O token identifica a sua empresa — todos os dados consultados e criados pertencem exclusivamente a ela. 

X-Api-Key: sk_live_SUA_CHAVE

Chamadas sem token, com token inválido ou com token inativado pelo contador retornam 401 Unauthorized.

Formato de erros

Erros retornam JSON com a mensagem no campo message:

{{ '{' }} "message": "Conta de debito '9.9.99' nao encontrada no plano de contas da empresa." {{ '}' }}
